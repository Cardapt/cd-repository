To clone a working copy for development, please clone one of the directories from this repository.
To do this, first you need to have a git client (http://git-scm.com/) then enter 'git clone https://github.com/Cardapt/cd-repository' in your git console.

For git starters: https://training.github.com/kit/downloads/github-git-cheat-sheet.pdf

Useful commands in git:
git status
git branch
git checkout [-b 'newbranchname']
git add [--all] [--ignore-removal]
git commit -am 'message'
git pull
git push
